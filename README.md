🛡️ SentinURL: Ultimate Phishing Detection System

SentinURL is a high-intelligence, multi-layered security engine designed to detect and block phishing attempts with extraordinary precision. Built for a Graduation Project, the system combines state-of-the-art Machine Learning with a dynamic "Self-Healing" immunization pipeline to adapt to new threats in real-time.
🚀 Key Highlights & Performance
*   🏆 99.88% Global Accuracy: Validated against a massive core dataset of 629,000+ unique URLs.
*   🧬 98.02% Zero-Day Detection: Exceptional performance on live, unseen malware extracted from URLHaus.
*   ⚡ Sub-100ms Inference: Optimized for real-time protection with a footprint that respects system resources.
*   🔄 Dynamic Learning: An automated retraining loop ("Golden Gate") that immunizes the system against missed threats.

 🧠 Advanced Dual-Stage Architecture
SentinURL doesn't just look for keywords; it understands the "DNA" of a URL using a hybrid approach:

1.  Stage 1: Lexical Intelligence (NLP)
    *   Uses TF-IDF Char N-Grams (2-6 chars) to identify malicious text signatures.
    *   Calibrated Logistic Regression provides a "Confidence Score" for every prediction.
2.  Stage 2: Structural Fingerprinting (HGB)
    *   Analyzes 29 mathematical features including entropy, subdomain depth, TLD risk, and brand deception tokens.
    *   Powered by a Histogram Gradient Boosting model for ultra-fast, non-linear classification.
3.  Hybrid Fusion: Predictions from both stages are weighted and fused to ensure maximum accuracy and near-zero false positives.

 🛡️ Multi-Layered Protection Strategy
SentinURL employs 10 Active Protection Layers:
*   Allowlisting: Trusted domain priorities (including Jordanian Institutional support).
*   Threat Intelligence: Integrated live feeds for known malware signatures.
*   Visual Similarity (Typosquatting): Detects "brand-lookalike" domains.
*   Machine Learning: The core 2-stage neural-style detection logic.
*   Local Policy: Ability for users to report and "Block" or "Allow" custom domains instantly.

 📊 Interactive Dashboard
A professional Streamlit UI provides:
*   Real-time Scan Engine: Check any URL with a detailed risk breakdown.
*   Analytics Hub: Visual statistics for detection rates and threat categories.
*   Report Center: Add new threats to the Master Dataset to trigger an immunization re-train.
*   System Health: Live monitoring of active protection layers.
