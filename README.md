## 🧠 Advanced Dual-Stage Architecture
SentinURL doesn't just look for keywords; it understands the "DNA" of a URL using a hybrid approach:

1.  **Stage 1: Lexical Intelligence (NLP)**
    *   Uses **TF-IDF Char N-Grams (2-6 chars)** to identify malicious text signatures.
    *   Calibrated Logistic Regression provides a "Confidence Score" for every prediction.
2.  **Stage 2: Structural Fingerprinting (HGB)**
    *   Analyzes **29 mathematical features** including entropy, subdomain depth, TLD risk, and brand deception tokens.
    *   Powered by a **Histogram Gradient Boosting** model for ultra-fast, non-linear classification.
3.  **Hybrid Fusion:** Predictions from both stages are weighted and fused to ensure maximum accuracy and near-zero false positives.
