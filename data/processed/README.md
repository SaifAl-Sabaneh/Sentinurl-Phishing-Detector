# SentinURL Processed Dataset

Due to GitHub's file size limitations, the complete processed machine learning dataset is securely hosted on Kaggle for interactive viewing and downloading.

* **Processed Dataset (Kaggle):** [View / Access SentinURL Processed Dataset on Kaggle](https://www.kaggle.com/datasets/saifsabaneh/sentinurl-processed-dataset-with-features)

### Description
This dataset contains the fully structured mathematical arrays utilized to train the SentinURL Machine Learning models. The raw malicious and benign URLs have been successfully passed through the automated SentinURL Data Extraction Pipeline, which parsed and extracted over 110 unique structural, lexical, and heuristic features for each row.
