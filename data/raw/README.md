# SentinURL Raw Dataset

Due to GitHub's file size limitations, the complete, un-extracted raw dataset is securely hosted on Google Drive.

* **Raw Dataset (Google Drive):** [View / Access SentinURL Raw Dataset.csv](https://docs.google.com/spreadsheets/d/1xO9nCFxRJRp3wBKRnPUMkHMfmXoSGCXp9UHjCN-brk4/edit?usp=sharing)

### Description
This dataset contains the original, un-extracted malicious and benign URLs gathered from verified threat intelligence feeds (PhishTank, OpenPhish) and the Tranco Top 1 Million list, prior to the SentinURL automated feature extraction pipeline.
